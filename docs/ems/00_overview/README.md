# EnergizeOS™ EMS — Public Documentation

Welcome to the **public documentation repository** for the EnergizeOS™ Energy Management System (EMS).

This repository provides **high-level, non-proprietary documentation** intended for external stakeholders, including:

- Customers
- EPC partners
- Utilities
- Auditors
- Investors
- Regulatory reviewers

---

## What You Will Find Here

- System positioning and scope definitions
- Conceptual architecture overviews
- Governance principles (non-confidential)
- Publicly shareable reference materials

---

## What You Will NOT Find Here

To protect system integrity and intellectual property, this repository does **not** include:

- Control algorithms
- Strategy parameters
- Relay logic or IO mappings
- Wiring diagrams
- FAT or commissioning scripts
- Internal engineering notes

---

## Authoritative Reference

The primary authoritative document in this repository is:

➡ **EMS Master Index (Public Edition)**

## Intended Use

This repository is designed to:

- Clarify system responsibilities
- Support technical discussions
- Enable compliant project integration
- Avoid ambiguity between EMS and protection systems

It is **not** intended to replace contractual documents or detailed engineering specifications.
